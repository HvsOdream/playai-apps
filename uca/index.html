자격증 개요 · 추천 학습 루트 · 출제 유형 안내
1 · UCA 자격증이란
Unity Certified Associate: Game Developer는 Unity 공식 입문~주니어 레벨 자격증입니다. Unity 에디터 활용, 프로그래밍 기초, 물리·애니메이션·UI·라이팅 등 게임 개발 전반과 게임 디자인·프로젝트 관리·취업 준비 같은 산업 지식까지 검증합니다.

· 형식: 객관식 중심 (이 앱의 4개 유형 — 객관식·핫스팟·드래그·매칭 — 이 실제 유형을 반영)
· 시간: 약 90분 내외 (회차·언어에 따라 다를 수 있음)
· 합격 기준: Unity 비공개 (통상 정답률 70% 이상을 목표로 준비 권장)
· 응시 방법: Unity 공식 인증 페이지에서 시험 바우처 구매 후 온라인 감독 시험 또는 시험 센터에서 응시
· 공식 정보: unity.com 인증 페이지 · Unity Learn 공식 무료 대비 코스
※ 문항 수·합격선은 Unity가 공식 공개하지 않으므로 위 공식 페이지 기준으로 최종 확인하세요.
2 · 추천 학습 루트 (이 앱 활용법)
STEP 1 — 진단: MODE 02 연습 모의고사 50문항을 먼저 풀어 현재 실력과 약점 도메인을 파악하세요.
STEP 2 — 약점 보강: 대시보드의 "약점 3개 집중 연습"과 "틀린 문제만 복습"으로 정답률 낮은 도메인부터 끌어올리세요. MODE 01에서 도메인·난이도를 직접 골라 연습할 수도 있습니다.
STEP 3 — 실전 감각: 연습 정답률이 70%를 넘으면 MODE 02를 반복해 시간 감각을 익히고, 교수님이 여는 MODE 03 시험 회차로 실전처럼 점검하세요.
목표선: 모의고사 정답률 80% 이상이 꾸준히 나오면 실제 응시 준비가 된 것입니다.
3 · 출제 비중 & 자주 나오는 유형
이 앱의 207문항은 공식 16개 도메인 비중(블루프린트)에 맞춰 구성되어 있으며, 모든 도메인이 최소 10문항 이상 연습 가능합니다. 특히 아래 영역이 비중이 크고 자주 출제됩니다:

· Editor Interface — 각 창(Hierarchy/Inspector/Project/Console/Scene/Game)의 역할, 창 화면에서 위치 찾기(핫스팟형)
· Programming — 생명주기 순서(Awake→OnEnable→Start→Update), Update vs FixedUpdate, Time.deltaTime, GetComponent
· Physics — Rigidbody 속성, Collider 종류·성능, isTrigger vs Collision, Raycast
· Animation — Animator 상태·전환·파라미터, Any State
· Asset·UI — Prefab 개념과 Overrides, Canvas Render Mode, Rect Transform·앵커

유형 팁: ① "~의 역할은?" 식 정의 문제가 가장 많음 ② 절차를 순서대로 배열하는 문제(순서형) ③ 개념-설명 짝짓기(매칭형) ④ 에디터 화면에서 위치 클릭(핫스팟형). 모두 이 앱에서 그대로 연습됩니다.
응시 팁: 실제 시험은 영어 출제가 기본이므로, 이 앱에서 익힌 개념의 영문 용어(Hierarchy, Rigidbody, Prefab, Animator Parameter 등)를 함께 기억해 두세요. 모르는 문제는 표시해 두고 넘어간 뒤 마지막에 돌아오는 전략이 유효합니다 — 이 앱의 시험 모드 문항 그리드가 같은 방식입니다.
※ 실제 기출문제 원문은 응시 규정(NDA)상 공개·복제가 금지되어 있어, 유형·경향 안내로 제공합니다.
